A modern terminal-styled quote generator built to master the integration between **React** and **Node.js**.

## 🚀 Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Axios.
- **Backend**: Node.js, Express, CORS.
- **API**: Integration with DummyJSON for data fetching.

## 🧠 Key Learnings

- **Fullstack Connectivity**: Linking a React frontend (Vite) to a custom Express REST API.
- **CORS Management**: Solving cross-origin security issues to allow port communication.
- **TypeScript Power**: Using `interfaces` to ensure data integrity across the stack.
- **State & Logic**: Handling loading, error, and conditional rendering with React Hooks.
- **Troubleshooting**: Managing port conflicts (`EADDRINUSE`) and directory path errors (`ENOENT`).

## 🛠️ How to Run

1. **Clone the repo**
2. **Backend**: `cd backEnd && npm install && node server.js` (Runs on port 3000)
3. **Frontend**: `cd frontEnd && npm install && npm run dev` (Runs on port 5173)
